# Wallpaper Gallery

A modern wallpaper website that lets users browse, download, and favorite wallpapers by category.
Features include:
- Search and category filter
- Random wallpaper of the day
- Favorites saved locally
- Dark/light mode toggle

## How to use
1. Open index.html in your browser.
2. Edit images.json to add new wallpapers.
3. Replace music.mp3 with your own background track.
